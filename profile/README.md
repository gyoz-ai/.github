### What is gyoza?

gyoza is an open-source browser extension powered by AI. Point it at any website, tell it what to do in plain language, and it handles the rest — navigating pages, clicking buttons, filling out forms, extracting data, and more.

- **Works on any website** — no setup or manifest needed
- **Multi-LLM** — choose Claude, OpenAI, or Gemini (bring your own key or use our managed proxy)
- **Recipe-powered** — community-built sitemaps give the AI a head start on popular sites
- **Privacy first** — your API key stays local, no server-side query storage

### The Extension

[**web-copilot**](https://github.com/gyoz-ai/web-copilot) is our flagship project — a browser extension that turns any website into an AI-navigable workspace. Install it on Chrome, Firefox, or Safari, point it at any page, and tell it what to do.

### How it works

```
1. Install the extension       → Chrome, Firefox, or Safari — one click
2. Add your API key or subscribe → BYOK for Claude/OpenAI/Gemini, or use our managed plan
3. Visit any website and ask   → Natural language prompt, AI handles the rest
```
